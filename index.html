🔄 Интеграция с 1С

🔑 Авторизация — Bearer Token
Все запросы к эндпоинту синхронизации требуют заголовок:
Authorization: Bearer YOUR_SECRET_TOKEN
Токен выдаётся администратором 5lb. Публичные GET-эндпоинты авторизации не требуют.
POST /api/catalog-5lb/sync Создать / обновить товар из 1С
Логика upsert: если товар с указанным articleMain уже существует — обновляется. Если нет — создаётся новый.

GET /api/catalog-5lb/by-article/:articleMain Товар по артикулу
Возвращает все варианты товара по основному артикулу. В ответе включён imageUrl с полным URL, а также imageWidth и imageHeight — реальные размеры изображения в пикселях.
Path-параметр
Параметр Тип Описание
articleMainДА
 string Основной артикул товара
Пример запроса
GET https://app.5lb.pro/api/catalog-5lb/by-article/1010002
Пример ответа
{
  "items": [
    {
      "id":           "abc123",
      "productName": "Whey Gold Standard",
      "articleMain": "1010002",
      "imageUrl":    "https://app.5lb.pro/images/catalog/1010002-CLYLW-VOL700.png",
      "imageWidth":  800,   // null если фото нет
      "imageHeight": 800,   // null если фото нет
      // ... остальные поля
    }
  ]
}
Коды ответа
200Товар найден
404Товар с таким артикулом не найден
GET /api/catalog-5lb/:id Товар по ID
Path-параметр
Параметр Тип Описание
idДА
 string Уникальный ID товара (CUID)
Пример запроса
GET https://app.5lb.pro/api/catalog-5lb/abc123
Возвращает один объект товара (не массив). При отсутствии — 404.
GET /api/catalog-5lb/sections Категории
Возвращает уникальные значения section1, section2, section3, section4 для построения дерева категорий.
Пример ответа
{
  "section1": ["Спортивное питание", "Витамины", "..."],
  "section2": ["Протеин", "Аминокислоты", "..."],
  "section3": ["..."],
  "section4": ["..."]
}
GET /api/catalog-5lb/brands Список брендов
Пример ответа
{ "brands": ["BSN", "Optimum Nutrition", "Scitec Nutrition", "..."] }
GET /api/catalog-5lb/countries Страны
Пример ответа
{ "countries": ["Германия", "Россия", "США", "..."] }
GET /api/catalog-5lb/flavors Список вкусов
Пример ответа
{ "flavors": ["Ваниль", "Клубника", "Шоколад", "..."] }
GET /api/catalog-5lb/stats Статистика каталога
Пример ответа
{
  "totalProducts":   150,
  "totalBrands":     25,
  "totalCountries":  8,
  "totalCategories": 12
}

🖼️ Изображения товаров

URL /images/catalog/{articleMain}-{articleAdditional}.png Фото товара
Формат имени файла:
articleMain-articleAdditional.ext — если доп. артикул есть
articleMain.ext — если доп. артикул пустой

В ответах API поле imageUrl содержит готовый полный URL или null.
Эндпоинт /by-article дополнительно возвращает imageWidth и imageHeight.
Примеры URL
https://app.5lb.pro/images/catalog/1010002-CLYLW-VOL700.png https://app.5lb.pro/images/catalog/1010002.png
