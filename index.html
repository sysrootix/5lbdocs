5LB API

Документация для разработчиков

Публичные REST API endpoints для работы с каталогом товаров. Все запросы — GET, авторизация не требуется.

Base URL https://app.5lb.pro

Быстрый доступ

📦
Каталог товаров
/api/catalog-5lb
📊
Статистика
/api/catalog-5lb/stats
🏷️
Категории
/api/catalog-5lb/sections
🌍
Бренды
/api/catalog-5lb/brands

Endpoints

GET /api/catalog-5lb Получить весь каталог
Параметры запроса
Параметр Тип Описание
page
необязательный
 integer Номер страницы. По умолчанию 1
limit
необязательный
 integer Товаров на странице. По умолчанию 20, максимум 100
search
необязательный
 string Поиск по названию, бренду, артикулу
section1
необязательный
 string Фильтр: основная категория
section2
необязательный
 string Фильтр: подкатегория
section3
необязательный
 string Фильтр: раздел 3
section4
необязательный
 string Фильтр: раздел 4
brand
необязательный
 string Фильтр по бренду
country
необязательный
 string Фильтр по стране производителя
Пример запроса
GET https://app.5lb.pro/api/catalog-5lb?page=1&limit=20&search=протеин

      "articleAdditional":"CLYLW-VOL700",
      "imageUrl":         "https://app.5lb.pro/images/catalog/1010002-CLYLW-VOL700.png",
      "weightVolume":     "2.27 кг",

      "color":            null,

    }
  ],
  "pagination": {
    "page":       1,
    "limit":      20,

  }
}
GET /api/catalog-5lb/sections Категории
Возвращает уникальные значения section1, section2, section3, section4 — все разделы каталога.
Пример запроса
GET https://app.5lb.pro/api/catalog-5lb/sections
GET /api/catalog-5lb/by-article/:articleMain Товар по артикулу
Path-параметр
Параметр Тип Описание
articleMain
 string Основной артикул товара
Пример запроса
GET https://app.5lb.pro/api/catalog-5lb/by-article/1010002
GET /api/catalog-5lb/:id Товар по ID
Path-параметр
Параметр Тип Описание
id
 string Уникальный ID товара в базе данных
Пример запроса
GET https://app.5lb.pro/api/catalog-5lb/abc123
GET /api/catalog-5lb/brands Список брендов
Возвращает отсортированный массив всех уникальных брендов.
Пример ответа
{
  "brands": ["BSN", "Optimum Nutrition", "Scitec Nutrition", "..."]
}
GET /api/catalog-5lb/countries Страны производителей
Возвращает отсортированный массив всех уникальных стран-производителей.
Пример ответа
{
  "countries": ["Россия", "США", "..."]
}
GET /api/catalog-5lb/flavors Список вкусов
Возвращает отсортированный массив всех уникальных вкусов.
Пример ответа
{
  "flavors": ["Ваниль", "Шоколад", "..."]
}
GET /api/catalog-5lb/stats Статистика каталога
Возвращает общую статистику по каталогу.
Пример ответа
{
  "totalProducts":   150,
  "totalBrands":     25,
  "totalCountries":  8,
  "totalCategories": 12
}

Изображения товаров

URL /images/catalog/{articleMain}-{articleAdditional}.png Изображение товара
Формат имени файла:
articleMain-articleAdditional.ext — если доп. артикул есть
articleMain.ext — если доп. артикул пустой

В ответе API есть поле imageUrl — готовый полный URL или null.
Пример URL
https://app.5lb.pro/images/catalog/1010002-CLYLW-VOL700.png
