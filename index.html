5LB API

Документация для разработчиков

Публичные REST API endpoints для работы с каталогом товаров. Все запросы — GET, авторизация не требуется.

Base URL https://app.5lb.pro

Быстрый доступ

/api/catalog-5lb
📊
Статистика

Категории

🌍
Бренды
/api/catalog-5lb/brands

Endpoints

GET /api/catalog-5lb Получить весь каталог
Query-параметры
Параметр Тип Default Описание
page
необязательный
 integer 1 Номер страницы
limit
необязательный
 integer 20 Товаров на странице, максимум 100
search
необязательный
 string — Поиск по названию, бренду, артикулу (без учёта регистра)
section1
необязательный
 string — Фильтр: основная категория (точное совпадение)
section2
необязательный
 string — Фильтр: подкатегория
section3
необязательный
 string — Фильтр: раздел 3
section4
необязательный
 string — Фильтр: раздел 4
brand
необязательный
 string — Фильтр по бренду
country
необязательный
 string — Фильтр по стране производителя
Примеры запросов
GET https://app.5lb.pro/api/catalog-5lb?page=1&limit=20&search=протеин
GET https://app.5lb.pro/api/catalog-5lb?section1=Спортивное питание&section2=Протеин&brand=Optimum Nutrition

GET /api/catalog-5lb/sections Категории
Возвращает уникальные значения section1, section2, section3, section4 — для построения дерева категорий.
Пример запроса
GET https://app.5lb.pro/api/catalog-5lb/sections
Пример ответа
{
  "section1": ["Спортивное питание", "Витамины и минералы", "Здоровое питание"],
  "section2": ["Аминокислоты", "Протеин", "Предтренировочные"],
  "section3": ["BCAA", "Глютамин"],
  "section4": []
}
GET /api/catalog-5lb/by-article/:articleMain Товар по артикулуобновлено
Теперь ответ включает imageWidth и imageHeight — реальные размеры изображения в пикселях, считанные с сервера. Если фото не загружено — поля равны null.
Path-параметр
Параметр Тип Описание
articleMain
 string Основной артикул товара. Может вернуть несколько вариантов (разные вкусы, фасовки).
Пример запроса
GET https://app.5lb.pro/api/catalog-5lb/by-article/1010002
Пример ответа
{
  "items": [
    {
      "id":                   "abc123",
      "productName":         "Whey Gold Standard",
      "brand":               "Optimum Nutrition",
      "articleMain":         "1010002",
      "articleAdditional":   "CLYLW-VOL700",
      "imageUrl":            "https://app.5lb.pro/images/catalog/1010002-CLYLW-VOL700.png",
      "imageWidth":          800,   // null если фото не загружено
      "imageHeight":         800,   // null если фото не загружено
      "weightVolume":        "2.27 кг",
      "servings":            "74",
      "flavor":              "Шоколад",
      "country":             "США",
      "releaseForm":         "Порошок",
      "section1":            "Спортивное питание",
      "section2":            "Протеин"
    }
  ]
}
Поля ответа
Поле Тип Описание
imageUrl string | null Полный URL изображения или null
imageWidth number | null Ширина изображения в пикселях
imageHeight number | null Высота изображения в пикселях
articleMain string Основной артикул
articleAdditional string | null Дополнительный артикул (вариация)
productName string Полное название товара
brand string | null Бренд
country string | null Страна производителя
flavor string | null Вкус
weightVolume string | null Вес / объём
servings string | null Количество порций
releaseForm string | null Форма выпуска
description string | null Описание
howToUse string | null Способ применения
ingredients string | null Состав
compositionPerServing string | null Состав на порцию (КБЖУ)
GET /api/catalog-5lb/:id Товар по ID
Возвращает один объект товара по его внутреннему ID базы данных. При отсутствии — 404.
Path-параметр
Параметр Тип Описание
id
 string Уникальный CUID (поле id из ответа каталога)
Пример запроса
GET https://app.5lb.pro/api/catalog-5lb/cmmbpn9g6000cc2opmo8shz83
Пример ответа
{
  "id":                   "cmmbpn9g6000cc2opmo8shz83",
  "productName":         "Whey Gold Standard",
  "brand":               "Optimum Nutrition",
  "articleMain":         "1010002",
  "imageUrl":            "https://app.5lb.pro/images/catalog/1010002-CLYLW-VOL700.png",
  // ... полный объект товара
}
GET /api/catalog-5lb/brands Список брендов
Возвращает отсортированный массив всех уникальных брендов каталога.
Пример запроса
GET https://app.5lb.pro/api/catalog-5lb/brands
Пример ответа
{
  "brands": ["BSN", "Dymatize", "Optimum Nutrition", "Scitec Nutrition", "..."]
}
GET /api/catalog-5lb/countries Страны производителей
Возвращает отсортированный массив всех уникальных стран-производителей.
Пример запроса
GET https://app.5lb.pro/api/catalog-5lb/countries
Пример ответа
{
  "countries": ["Германия", "Россия", "США", "..."]
}
GET /api/catalog-5lb/flavors Список вкусов
Возвращает отсортированный массив всех уникальных вкусов.
Пример запроса
GET https://app.5lb.pro/api/catalog-5lb/flavors
Пример ответа
{
  "flavors": ["Ваниль", "Клубника", "Натуральный", "Шоколад", "..."]
}
GET /api/catalog-5lb/stats Статистика каталога
Возвращает общую статистику по каталогу.
Пример ответа
{
  "totalProducts":   150,
  "totalBrands":     25,
  "totalCountries":  8,
  "totalCategories": 12
}

Изображения товаров

URL /images/catalog/{articleMain}-{articleAdditional}.png Изображение товара
Формат имени файла:
articleMain-articleAdditional.ext — если доп. артикул есть
articleMain.ext — если доп. артикул пустой

В ответе API поле imageUrl уже содержит готовый полный URL или null.
Эндпоинт /by-article дополнительно отдаёт imageWidth и imageHeight.
Примеры URL
https://app.5lb.pro/images/catalog/1010002-CLYLW-VOL700.png
https://app.5lb.pro/images/catalog/1010002.png
